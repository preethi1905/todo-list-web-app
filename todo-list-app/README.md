# 🌈 Smart To-Do List Web App

## 📌 Project Overview

This project is a dynamic and interactive To-Do List Web Application developed using HTML5, CSS3, and Vanilla JavaScript. It allows users to efficiently manage daily tasks by adding, completing, and deleting tasks in real time. The application features a colorful and responsive user interface, providing a smooth user experience across desktop and mobile devices.

## 🎯 Objective

The objective of this project is to design and develop a To-Do List application that includes:

* Add new tasks dynamically
* Mark tasks as completed
* Delete individual tasks
* Store tasks using Local Storage
* Display task count
* Dark/Light mode toggle
* Responsive design for all devices

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension (if not already installed).
4. Right-click on `index.html`.
5. Select **Open with Live Server**.
6. Start managing your daily tasks through the application.
